Library List in ILEditor

The library list will apply to every command sent from ILEditor, including all compiles. To open the library list editor:

  1. Select the Tools dropdown
  2. Select Library List

You can use the Ctrl+L shortcut to open the same window.

Editing the list

To add an item to the list, put the library name in the textbox to the right and then select "Add library" - it will then appear on the list.

To remove an item from the list, select the item in the list and press the Delete key on your keyboard.

To move an item up or down in the list, select item and use the Up and Down arrow keys on your keyboard.
