Opening a specific member

  1. Select the File dropdown
  2. Select Open
  3. Select Member

Or use the Ctrl+O shortcut to open the same window.

Opening a member via the member listing

If the Toolbox is not open, select Tools->Open Toolbox.

  1. Select Member Browser from the Toolbox
  2. Enter in your library and source-physical file, followed by pressing the fetch button to the right of the text boxes
  3. When the list has loaded, double click the member name you'd like to open.
